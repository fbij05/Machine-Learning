# Machine-Learning

This repository contains lab submissions for the Machine Learning course. All labs are provided as Jupyter Notebooks (.ipynb) for easy execution and understanding.

# Repository Structure

Each lab is organized in its own folder (e.g., Lab1/, Lab2/, etc.)

Each folder contains the corresponding Jupyter Notebook and any supporting files

# Requirements

Python 3.x

Jupyter Notebook

Libraries: numpy, pandas, scikit-learn, matplotlib

# Notes

All code is commented for clarity.

Files are organized by lab to make navigation easier.
